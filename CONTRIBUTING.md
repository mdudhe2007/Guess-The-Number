# Contributing to Guess The Number

Thank you for considering contributing! 🎉 All contributions — bug fixes, features, or docs — are welcome.

---

## 🛠️ How to Contribute

1. **Fork** the repository
2. **Create a branch** for your change
   ```bash
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** and test them locally
4. **Commit** with a clear message
   ```bash
   git commit -m "feat: add difficulty levels"
   ```
5. **Push** to your fork and open a **Pull Request**

---

## 💡 Ideas for Contributions

- Add difficulty levels (Easy / Medium / Hard with different ranges)
- Add a high-score / best-attempts tracker
- Add a replay prompt after winning
- Port to a GUI using a library like SFML or Qt
- Write unit tests

---

## 📐 Code Style

- Use **4-space indentation**
- Keep variable names descriptive
- Add comments for non-obvious logic
- Compile cleanly with `-Wall -Wextra` before submitting

---

## 🐛 Reporting Bugs

Open an [Issue](../../issues) with:
- Steps to reproduce
- Expected vs actual behavior
- Compiler version and OS
